## はじめに
これは「立体４目並べ」ゲームのリポジトリです。

## コンセプト
思考ルーチンを学習するための枠組みとして「立体４目並べ」を制作する。

## 仕様
- 思考ルーチン（COM）を実装
- Mini-Max法、αβ法を用いて思考ルーチンを設計
- 最大探索局面数：１００万局面
- レベル星６の初期状態では自分の手から数えて６手先まで探索する

## ルール
- 平面、立体それぞれの縦、横、斜めのいずれかを自分の色でそろえると勝利
